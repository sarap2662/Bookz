# PROJECT BOOKS-API

## Routes Table

| METHOD |    PATH    |                  PURPOSE |
| ------ | :--------: | -----------------------: |
| GET    |     /      |                Home Page |
| GET    |   /books   |       Pulls up all books |
| GET    | /books/:id | Pulls up a specific book |
| PUT    | /books/:id |           Updates a book |
| DELETE | /books/:id |           Deletes a book |
| POST   |   /books   |       Creates a new book |
